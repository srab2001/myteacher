# Signature Module Wireframe

## Overview
The Signature Module handles electronic signature collection for IEP/504/BIP documents. It tracks signature requests, captures signatures, and validates that all required parties have signed.

## Business Rules

1. Required signers must all sign before request is COMPLETED
2. Parent/Guardian signature is always required for IEP
3. Signature requests expire after configurable period (default 30 days)
4. Reminders sent automatically at 7, 3, and 1 day before due date
5. Declined signatures trigger notification to case manager
6. Signatures include legal acknowledgment checkbox
7. All signature actions logged to Decision Ledger
8. Verification code is single-use (invalidated after signing)
9. Signatures cannot be revoked after submission
10. PDF regenerated with signature page when all signatures obtained
